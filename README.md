> A Python Tkinter application that simulates a micromouse navigating a maze using the **Flood Fill Algorithm**.  
> Users can design their own maze, set start and end points, and watch the mouse find the optimal path in real-time.

---

## ðŸ”Ž Overview

This interactive application allows you to:

- Create custom mazes of any size.
- Set **start** and **end points** for the micromouse.
- Toggle walls and open paths interactively.
- Visualize the **Flood Fill Algorithm** as the mouse calculates distances and moves toward the goal.
- Reset the maze and try different configurations.

Perfect for **learning pathfinding algorithms** and **visualizing maze-solving strategies**.
